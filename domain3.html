Domain 3: Security Architecture (18%)

This domain covers secure network and system architecture, secure design principles, and physical security controls.

Key Topics

  • Network Security (Zero Trust, Defense in Depth, Segmentation)
  • Secure Systems Design
  • Secure Staging and Deployment
  • Physical Security Controls
  • Embedded and Specialized Systems

Study Notes

  • Zero Trust: Never trust, always verify.
  • Defense in Depth: Multiple layers of security.
  • System Hardening: Secure configuration of systems.
  • Physical Security: Access control, surveillance, and environmental controls.
