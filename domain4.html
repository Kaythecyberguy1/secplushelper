Domain 4: Security Operations (28%)

This domain covers security monitoring, incident response, vulnerability management, and automation in security operations.

Key Topics

  • Security Monitoring (SIEM, EDR, Network Monitoring)
  • Incident Response Procedures
  • Vulnerability Management
  • Automation and Orchestration
  • Identity and Access Management Controls

Study Notes

  • SIEM: Security Information and Event Management for log analysis and alerting.
  • Incident Response: Preparation, identification, containment, eradication, recovery, and lessons learned.
  • Vulnerability Management: Scanning, patching, and configuration management.
  • Automation: Using scripts and tools to streamline security operations.
