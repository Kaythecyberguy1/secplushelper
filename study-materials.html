Security+ SY0-701 Study Materials

Below are the official exam domains and their key topics. Click on each domain to expand and view detailed study materials.

Domain 1: General Security Concepts (12%)

Key Topics:

  • Compare and contrast various types of security controls
  • Explain the importance of applicable regulations, standards, or frameworks
  • Explain the importance of policies to organizational security
  • Summarize risk management processes and concepts

Study Notes:

Security Controls
  • Preventive controls: Stop incidents from occurring
  • Detective controls: Identify incidents after they occur
  • Corrective controls: Restore systems after incidents
  • Deterrent controls: Discourage potential attackers
Risk Management
  • Risk = Threat × Vulnerability × Impact
  • Risk assessment: Identify, analyze, and evaluate risks
  • Risk treatment: Accept, transfer, mitigate, or avoid

Domain 2: Threats, Vulnerabilities, and Mitigations (22%)

Key Topics:

  • Compare and contrast common threat actors and motivations
  • Explain common threat vectors and attack surfaces
  • Explain various types of vulnerabilities
  • Given a scenario, analyze indicators of malicious activity

Study Notes:

Threat Actors
  • Nation-state actors: Government-sponsored attacks
  • Hacktivists: Attack for political or social causes
  • Organized crime: Attack for financial gain
  • Script kiddies: Use existing tools without deep knowledge
Common Attack Vectors
  • Email phishing and spear phishing
  • Malware and ransomware
  • Social engineering attacks
  • Zero-day exploits

Domain 3: Security Architecture (18%)

Key Topics:

  • Compare and contrast security implications of different architecture models
  • Given a scenario, implement secure network architecture concepts
  • Given a scenario, implement secure systems design
  • Explain the importance of secure staging deployment concepts

Study Notes:

Network Security
  • Zero Trust Architecture principles
  • Defense in depth strategy
  • Network segmentation and micro-segmentation
  • Secure protocols and encryption
Secure Systems Design
  • Least privilege principle
  • Secure defaults
  • Fail secure/fail safe
  • Separation of duties

Domain 4: Security Operations (28%)

Key Topics:

  • Given a scenario, implement security monitoring
  • Given a scenario, implement incident response procedures
  • Given a scenario, implement vulnerability management
  • Explain the importance of automation and orchestration

Study Notes:

Security Monitoring
  • SIEM systems and log analysis
  • Network monitoring tools
  • Endpoint detection and response (EDR)
  • Security information and event management
Incident Response
  • Preparation phase
  • Identification phase
  • Containment phase
  • Eradication phase
  • Recovery phase
  • Lessons learned phase

Domain 5: Security Program Management and Oversight (20%)

Key Topics:

  • Given a scenario, implement security awareness practices
  • Explain the importance of policies, plans, and procedures
  • Given a scenario, implement security assessment activities
  • Explain the importance of third-party risk management

Study Notes:

Security Awareness
  • Phishing awareness training
  • Password security best practices
  • Social engineering awareness
  • Physical security awareness
Security Assessment
  • Vulnerability scanning
  • Penetration testing
  • Security audits
  • Compliance assessments
